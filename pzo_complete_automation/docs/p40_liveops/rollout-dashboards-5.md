# LiveOps Control Plane - Rollout Dashboards v5

## Overview

The LiveOps Control Plane (LCP) Rollout Dashboards v5 provide a comprehensive view of various aspects related to rollouts in your system. This document outlines the key features and functionalities available in this version of the dashboard.

## Key Features

1. **Rollout Overview:** A high-level summary of all active and completed rollouts across different services, including details like status, traffic splits, error rates, and more.

2. **Service Rollout Details:** Detailed insights into each service's rollout, showing metrics such as traffic distribution, error rates, and user feedback.

3. **Rollout Comparison:** Compare the performance of two different versions (canaries) of a service to identify any differences.

4. **Real-time Metrics:** Monitor real-time data for key performance indicators (KPIs), such as request rate, error rate, and latency.

5. **Service Health:** Visual representation of service health status based on various KPIs, helping to quickly identify any potential issues.

6. **Rollout Analytics:** Trend analysis over time for selected rollouts, including detailed charts and tables to aid in decision-making.

7. **User Feedback Integration:** Incorporate user feedback collected from various sources (e.g., Sentry, GitHub Issues) to assess the impact of rollouts on user experience.

## Getting Started

To access Rollout Dashboards v5, navigate to the LiveOps Control Plane web interface and look for the "Rollout Dashboards" option in the main menu. Detailed instructions for setup and configuration can be found in the [LiveOps Control Plane user guide](docs/p40_liveops/user-guide.md).
