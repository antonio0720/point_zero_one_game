# LiveOps Control Plane - Rollout Dashboards v1.5

## Overview

The `Rollout Dashboards v1.5` is a component of the LiveOps control plane, providing a user-friendly interface to monitor and manage rollouts effectively.

## Features

- **Rollout Overview:** A summary of all ongoing and completed rollouts across different services.
- **Service Detail View:** An in-depth look at each service's current rollout status, metrics, and logs.
- **Rollout Details:** Comprehensive information about the progress, traffic distribution, and errors for a specific rollout.
- **Error Analysis:** Identify issues in your rollouts by analyzing error rates and trends over time.
- **Alerting System:** Receive notifications when predefined conditions are met during your rollouts.

## Usage

Access the Rollout Dashboards via the LiveOps control plane interface. Navigate through the intuitive UI to manage, monitor, and analyze your rollouts with ease.

## Updates in v1.5

- **Enhanced Metrics:** New and improved metrics for better insights into rollout performance.
- **Improved Alerting System:** Customizable alert rules and enhanced notifications.
- **Error Trend Analysis:** Visualize error trends across multiple rollouts for quick identification of issues.
- **Log Integration:** Seamless integration with log management systems for easier debugging.

For more information, refer to the [LiveOps control plane documentation](docs/p40_liveops/liveops-documentation.md).
