# OBS Integration for Desktop Client (Version 9)

## Overview

This document outlines the integration of the Open Broadcaster Software (OBS) with our Desktop client in version 9. The goal is to provide a seamless workflow for live streaming and recording.

## Compatibility

The OBS integration in this version is compatible with OBS Studio versions 26.x, 27.x, and the latest stable release of OBS Studio (at the time of writing).

## Installation

1. Download the latest version of our Desktop client from [here](url_to_download_the_desktop_client).
2. Install the Desktop client following the instructions provided during setup.
3. Ensure you have OBS Studio installed on your system. If not, download it from the official website: https://obsproject.com/download
4. Launch both the Desktop client and OBS Studio.

## Configuration

1. In the Desktop client, navigate to `Settings` > `Integration` > `OBS Integration`.
2. Click on `Configure OBS`. A new window will open with your OBS studio path pre-filled.
3. Verify that the correct OBS Studio installation directory is selected, and click `Save & Close`.
4. If you're prompted to restart either the Desktop client or OBS Studio for the changes to take effect, comply with the request.

## Usage

1. In the Desktop client, navigate to the streaming/recording settings by clicking on the respective icons in the top-right corner.
2. You'll find an option to configure your stream/record settings using OBS Studio. Select it.
3. A new window will open displaying the OBS Studio settings interface. Here, you can customize various streaming and recording parameters such as resolution, bitrate, and encoder settings.
4. Once done, click `Apply` and then `OK` to save your changes in OBS Studio.
5. Back in the Desktop client, click on `Start Streaming/Recording` to initiate your stream or record using the configured OBS Studio settings.

## Troubleshooting

- If you encounter issues with the integration, ensure that both the Desktop client and OBS Studio are updated to their respective latest versions.
- If you're still facing problems, consult our support team at [support_url](url_to_support).

## Conclusion

With this OBS integration, our Desktop client now offers more flexibility for customizing your streaming and recording settings while maintaining a user-friendly interface. Enjoy live streaming and recording with enhanced quality!
