# Clip Studio 15 Desktop Client Documentation

## Overview

Clip Studio 15 is a powerful digital art software designed for illustrators, manga artists, and comic creators. This document provides a comprehensive guide to the desktop client version of Clip Studio 15.

## Installation

### System Requirements

- **Operating Systems:** Windows 7 or later, macOS X 10.13 or later
- **Processor:** Intel Core i5 or equivalent
- **RAM:** 4GB or more (8GB recommended)
- **Graphics Card:** OpenGL 2.0 or later compatible graphics card with at least 1 GB of dedicated VRAM

### Download and Installation

1. Visit the official Clip Studio website: https://www.clip-studio.com/en
2. Click on 'Download' to get the installer for your operating system.
3. Run the downloaded installer, follow the instructions to complete the installation process.

## Getting Started

### Launching Clip Studio 15

After successful installation, locate the Clip Studio 15 icon on your desktop or start menu and double-click it to launch the application.

### Welcome Screen

Upon launching, you will be greeted by the Welcome Screen where you can create a new project, open an existing one, access tutorials, or update the software.

## Interface Overview

Clip Studio 15 features a user-friendly interface with several key components:

1. **Toolbar:** Houses various drawing tools such as brushes, pens, and erasers.
2. **Layer Palette:** Allows you to manage layers for better organization of your artwork.
3. **Properties Palette:** Adjusts settings for the currently selected tool or layer.
4. **Tool Assistance Line:** Provides real-time assistance for tools like perspective, symmetry, and rulers.
5. **Tool Assistant Palette:** Offers various assistants to aid in your drawing process.
6. **Document Window:** Displays your artwork as you create it.
7. **Palettes Dock:** Holds additional palettes such as color, reference, and materials.
8. **Status Bar:** Shows useful information like the current brush size, layer mode, and file name.

## Creating a New Project

1. On the Welcome Screen, click 'New' or press `Ctrl+N` (Windows) / `Cmd+N` (Mac).
2. Select the desired template, format, or preset for your project.
3. Give your project a name and choose the location to save it.
4. Click 'OK' to create the new project and open it in the Clip Studio 15 interface.

## Conclusion

This guide provides an introduction to using Clip Studio 15 on the desktop. For more detailed information about specific features, tools, or techniques, we encourage you to explore the software further or consult the official Clip Studio documentation.
