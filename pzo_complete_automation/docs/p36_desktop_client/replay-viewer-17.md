Replay Viewer - v17
===================

Overview
--------

The Replay Viewer is a desktop application for visualizing and analyzing game replays of the popular video game, _Genshin Impact_. This version (v17) introduces several new features and improvements.

New Features
------------

* **Support for Version 2.5**: The Replay Viewer now supports the latest version of Genshin Impact (version 2.5). Users can load and analyze replays from this update seamlessly.
* **Improved Camera Controls**: The camera controls have been revamped to provide a more intuitive and responsive experience. Users can now smoothly zoom, pan, and rotate the camera during playback.
* **Character Stat Display**: A new feature that shows character stats (HP, ATK, DEF, etc.) during gameplay. This allows users to better understand how characters perform in specific situations.
* **Improved UI Scalability**: The user interface has been optimized for better scalability, ensuring a pleasant experience on various screen resolutions.

Installation
------------

To download the latest version of the Replay Viewer, visit our [GitHub releases page](https://github.com/replay-viewer/replay-viewer/releases). Choose the appropriate installer for your operating system and follow the installation instructions provided.

Usage
-----

1. Open the Replay Viewer application on your desktop.
2. Click on the "Open Replay" button, and select the replay file you wish to analyze.
3. The replay will be loaded, and you can use the camera controls to navigate through the gameplay.
4. Press the "Play" button to start playing the replay. Character stats will be displayed during gameplay if enabled in the settings.
5. Utilize various other features such as slow motion, fast forward, and frame-by-frame playback for detailed analysis.
