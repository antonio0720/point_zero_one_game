# Moderation-17

## Overview

This document provides a comprehensive guide for using the Moderation-17 feature in the admin console.

## Feature Description

Moderation-17 is a powerful tool designed to assist moderators in managing content within the system. It offers advanced filtering, approval/rejection capabilities, and detailed reporting.

## Getting Started

### Access Moderation-17

1. Log into the admin console with your credentials.
2. Navigate to the 'Moderation' section on the left-hand menu.
3. Select 'Moderation-17' from the sub-menu.

### Understanding the Interface

The Moderation-17 interface consists of several key components:

- **Content List**: Displays a list of pending content that requires moderation. Each row represents a piece of content with relevant details such as title, creator, and submission date.
- **Filters**: Allows you to refine the content list based on specific criteria like content type, creation date, or user.
- **Actions**: Provides options to approve, reject, or take additional actions on individual pieces of content.
- **Reporting**: Generates detailed reports about moderation activities, including approval rates, most active users, and trending content topics.

## Using Moderation Tools

### Approving Content

1. Locate the piece of content you wish to approve in the content list.
2. Click on the 'Approve' button for that content. The content will be moved out of the pending queue and made available for use within the system.

### Rejecting Content

1. Find the content you wish to reject in the content list.
2. Click on the 'Reject' button for that content. A popup window will appear, allowing you to provide a reason for rejection. Enter the required information and click 'Submit'. The content will be moved out of the pending queue and marked as rejected.

### Using Filters

1. To filter the content list, use the dropdown menus located above the content list. You can filter based on content type, user, creation date, or other relevant criteria.
2. Apply multiple filters by selecting combinations of options from each dropdown menu.

## Generating Reports

### Accessing Reporting

1. Navigate to the 'Reports' tab in the Moderation-17 interface.
2. Choose the type of report you wish to generate (e.g., approval rates, most active users, trending content topics).
3. Customize your report by selecting specific date ranges or filtering results based on user or content type.
4. Click 'Generate Report' to create and view the report.

## Troubleshooting

In case of any issues with Moderation-17, please consult the admin console support team for assistance.
