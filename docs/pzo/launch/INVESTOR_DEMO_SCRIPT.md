# Investor Demo Script

## Step 1: Introduction (30 seconds)

*   Welcome to Point Zero One Digital's demo script for investors.
*   This walkthrough will showcase our engine running and key features of our platform.

## Step 2: Engine Running (1 minute)

*   Open a terminal and navigate to the project root directory.
*   Run `npm run demo` to start the engine.
*   Verify that the engine is running by checking for console output.

## Step 3: Browser UI (30 seconds)

*   Open a web browser and navigate to `http://localhost:3000`.
*   Verify that the browser UI is displayed correctly, including navigation menu and main content area.

## Step 4: Play Three Cards (1 minute)

*   In the browser UI, click on the "Play" button to start playing cards.
*   Select three random cards by clicking on them in the card grid.
*   Verify that the selected cards are displayed correctly in the main content area.

## Step 5: Trigger Gamma Squeeze (30 seconds)

*   In the browser UI, click on the "Gamma Squeeze" button to trigger the event.
*   Verify that the engine responds correctly by checking for console output and UI updates.

## Step 6: Show Proof Card (30 seconds)

*   After triggering the Gamma Squeeze, verify that a proof card is displayed in the main content area.
*   Check that the proof card contains the expected information.

## Step 7: Show Leaderboard (30 seconds)

*   In the browser UI, click on the "Leaderboard" button to display the leaderboard.
*   Verify that the leaderboard is populated with the correct data and sorted correctly.

## Step 8: Show Daily Gauntlet CTA (30 seconds)

*   In the browser UI, verify that a call-to-action (CTA) for the daily gauntlet is displayed in the main content area.
*   Check that the CTA contains the expected information and links to the correct page.

## Step 9: Conclusion (1 minute)

*   Congratulations! You have completed the investor demo script walkthrough.
*   This concludes our demonstration of Point Zero One Digital's engine running and key features of our platform.
