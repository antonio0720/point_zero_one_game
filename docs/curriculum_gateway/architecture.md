# PZO Curriculum Gateway — Architecture

        ## Overview
        The curriculum gateway is the B2B API surface for institutional delivery of Point Zero One.
        It fronts four domain subsystems: licensing control plane, curriculum services, content ops,
        and trust/proof verification.

        ## Domain: `https://pointzeroonegame.com/api/v1/curriculum`

        ## Architecture Decisions
        - **Auth:** Unified JWT (RS256 production, HS256 dev)
        - **Validation:** Ajv (compiled JSON Schema)
        - **Tenancy:** Row-level isolation, institution_id on every table
        - **Events:** BullMQ queues for async operations
        - **Billing:** Stripe subscriptions for institutional licensing
        - **Observability:** Structured JSON logging, request correlation IDs

        ## Adapter Boundaries
        - Gateway routes never touch database directly
        - All database access goes through typed adapters
        - Adapters enforce tenant context on every query
        - Circuit breakers protect against downstream failures
