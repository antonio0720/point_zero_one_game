# Season 0 Promise, Scarcity Rules, and User-Visible Contract Copy

## Overview

In this document, we outline the core promises, scarcity rules, and user-visible contract copy for Point Zero One Digital's Season 0. This information will be utilized across landing pages, onboarding processes, profiles, and sharing features.

## Non-negotiables

1. **Financial Roguelike Game**: Point Zero One Digital is a 12-minute financial roguelike game that offers an immersive, strategic gaming experience with a unique twist of financial management.

2. **Sovereign Infrastructure Architect Design**: Our infrastructure is designed to be production-grade and deployment-ready, ensuring seamless gameplay for all users.

3. **TypeScript Coding Standards**: All code adheres to strict TypeScript standards, with 'any' never used in our codebase.

4. **Deterministic Effects**: All effects within the game are deterministic, ensuring fairness and reproducibility of outcomes.

## Implementation Spec

### Season 0 Promise

- **Gameplay Experience**: Engage in a thrilling 12-minute financial roguelike game that challenges your strategic thinking and decision-making skills.
- **Sovereign Infrastructure**: Benefit from our production-grade, deployment-ready infrastructure designed for optimal performance.
- **Deterministic Gameplay**: Experience fairness and reproducibility in every game session due to deterministic effects.

### Scarcity Rules

- **Limited Edition**: Season 0 offers a limited number of game copies, ensuring exclusivity and scarcity.
- **Non-Fungible Tokens (NFTs)**: Each game copy is represented by a unique NFT, verifiable on the blockchain.
- **Rarity Levels**: Game copies are divided into different rarity levels, with higher rarities offering additional in-game benefits and perks.

## Edge Cases

### Rarity Level Determination

- Each game copy is assigned a rarity level based on the order of purchase. The first purchased copies will have the highest rarity, followed by subsequent purchases in descending order.
