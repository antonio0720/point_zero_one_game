# SEO & Open Graph Spec for Point Zero One Digital - 12-minute financial roguelike game

Overview:
This document outlines the SEO title/description, OG image fields, canonical URL, and share-card 'learn more' link specifications for Point Zero One Digital's website.

Non-negotiables:
1. Strict adherence to the specified values in all marketing channels.
2. Use of precise, execution-grade language.
3. Zero fluff or bureaucratic jargon.
4. All code is strict-mode and never uses 'any' in TypeScript.
5. All effects are deterministic.

Implementation Spec:

### SEO Title
The SEO title for Point Zero One Digital should be: `Point Zero One Digital - 12-minute financial roguelike game`

### Description
The description should be a concise summary of the game and company, such as: `Experience a unique blend of strategy and finance in our 12-minute roguelike game. Sovereign infrastructure architect design. Production-grade, deployment-ready. Never use 'any' in TypeScript. All code is strict-mode. All effects are deterministic.`

### OG Image Fields
The Open Graph image fields should include:
1. `og:image`: The URL of the primary image to be used when sharing the game on social media platforms. Recommended size: 1200x630 pixels.
2. `og:image:alt`: Alternative text for the OG image, describing its content for screen readers and search engines.

### Canonical URL
The canonical URL is the primary URL of a webpage, indicating to search engines which version of the page is the original and authoritative one. For Point Zero One Digital, the canonical URL should be: `https://pointzeroonedigital.com`

### Share-Card 'Learn More' Link
The share-card 'learn more' link should direct users to the main integrity page (`/integrity`) for detailed information about the game and company.
